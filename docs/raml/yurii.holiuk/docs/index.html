/admin

/admin/tests post

post

post /admin/tests

  • Request
  • Response

Query Parameters

  • token: required (string)

Body

Media type: application/json

Type: TestHeader

Properties
  • questions: required (array of Question)

    Items: Question

    • id: required (number)
    • questionType: required (one of text, image, video)
    • answerType: required (one of oneChoice, multiplyChoice, openAnswer, yesNo)
    • question: required (string)
    • required: required (boolean)
    • answers: required (array of Answer)

      Items: Answer

      • id: required (number)
      • answer: required (string)

      Example:

      {
  "id": 5,
  "answer": "Colorado state"
}

Example:

{
  "id": 1,
  "name": "JS basic test",
  "questions": [
    {
      "id": 5,
      "question": "How old are you?",
      "questionType": "text",
      "required": true,
      "answerType": "oneChoice",
      "answers": [
	 {
	   "id": 12,
	   "answer": "Less then 20"
	 },
	 {
	   "id": 13,
	   "answer": "More then 20"
	 }
      ]
    }
  ]
}
